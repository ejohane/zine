# iOS Simulator Skill

This skill provides optimized scripts for iOS Simulator automation when using
the ios-simulator-mcp toolset.

## Location

- Canonical docs: `.opencode/skill/ios-simulator-skill/SKILL.md`
- Installed skill (Claude Code): `~/.claude/skills/ios-simulator-skill/`

## Key Scripts

- `screen_mapper.py` - Map UI elements with summarized output
- `navigator.py` - Semantic element finding and interaction
- `simctl_boot.py` / `simctl_shutdown.py` - Simulator lifecycle
- `accessibility_audit.py` - Accessibility audit checks
- `build_and_test.py` - Build app and run tests
